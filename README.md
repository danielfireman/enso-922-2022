# Bem-vindos(a) ao repositório da 922
Visando a manutenção do caráter prático da disciplina teremos um software-alvo, que estaremos desenvolvendo durante a disciplina. Iremos construir uma API HTTP (inspirada em REST) que calcula a raiz quadrada.

Além dos requisitos funcionais que serão levantados pelos grupos, devemos atender os seguintes requisitos não funcionais:
- Código fonte livre e aberto
- Licença MIT (permissiva)
- Usando o repositório online
- Usando integração e entrega contínuas
